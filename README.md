# Full-Stack Blog Platform

This repository contains a full-stack blog application built using the MERN stack (MongoDB, Express, React, Node.js) with Vite as the build tool for a streamlined development experience.

## Features

- **Authentication System:** Secure login and signup functionalities, including Google authentication.
- **JWT for Security:** Implementation of JSON Web Tokens for secure data transfer and session management.
- **Password Hashing:** Enhances security by hashing user passwords.
- **Dual Theme:** Dark and light themes to suit user preferences.
- **Search Functionality:** Allows users to search for blog posts.
- **Responsive Admin Dashboard:** For managing posts and user accounts effectively.

## Prerequisites

- Node.js
- MongoDB
- npm or yarn
